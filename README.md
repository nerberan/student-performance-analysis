# 📊 Анализ успеваемости студентов

## Описание
Программа анализирует данные об успеваемости студентов из CSV-файла.

## Функции
- Расчет средних оценок по предметам
- Общая успеваемость каждого студента
- Поиск лучшего и худшего студента
- Анализ связи посещаемости и успеваемости

## Запуск
```bash
python analysis.py
